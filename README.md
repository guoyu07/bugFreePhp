# BugFree
佛祖保佑，永无bug！：） just for fun !


# 使用方法 
## 1.composer 引入 
 
    composer require leon0204/bugfree dev-master 

## 2.在文件中引入 
    
    use BugFree\Utils as Utils;
    
## 3.为filepath添加佛祖庇护：） 
    
    $res = Utils\AddNote::bugfree($filepath);
    
# 注意

### 请确保文件有足够的sudo权限
